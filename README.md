# Dark-Forest---RPG-Game
A text-based RPG adventure created in Python

This project was created as a learning project to practice programming concepts, game logic design, object management, random events, and user interaction.

## Author

Author: Szymon Pacaj

## About the Project

Dark Forest is a simple text-based RPG game where the player explores dangerous areas, fights monsters, collects resources, upgrades equipment, and develops their character.

The main goal of the game is to reach level 20 and defeat the final boss.

The project focuses on learning Python programming and creating a larger application with multiple connected systems.

## Features

- Character progression system
- Experience and level system
- Turn-based combat
- Different types of enemies:
  - Common monsters
  - Rare monsters
  - Legendary monsters
  - Final boss
- Inventory system
- Healing and special potions
- Gold economy
- Shop system
- Resource gathering
- Mining system
- Random loot and treasure chests
- Crafting system with a blacksmith
- Weapons, armor, and tools upgrades

## Gameplay

At the beginning of the game, the player creates a character and starts exploring the Dark Forest.

The player can:

- Fight monsters to gain XP and gold
- Collect crafting materials
- Buy and sell items
- Upgrade weapons and armor
- Create tools using collected resources
- Open treasure chests
- Prepare for the final battle

## Requirements

- Python 3.10 or newer

No external libraries are required.

The project uses only built-in Python modules.

## Installation

Download the project files and open the project folder.

Run the game:

python main.py

## Controls

The game is controlled using the keyboard.

During gameplay, choose an option by entering the corresponding number.

Example:

1. Go to the forest
2. Go to the mine
3. Inventory
4. Shop
5. Blacksmith
